Bug Report

CSS Styling changes OpenType substitution

Problem

A string has two characters:

A B

If the characters render next to each other the first character is replaced by the ligature feature of the font. See OpenType Docs (Microsoft) – Registered features.

AB

If the string gets formated in CSS with 

            
p::first-letter { color: red; }
your current browser renders like that:

AB

Expected:

—B

Renderings

Incorrect rendering

Google Chrome 69.0.3497.100 on MacOS 10.14 Safari 12.0 (14606.1.36.1.9) on MacOS 10.14 Apple TextEdit 1.14 (340) on MacOS 10.14

Correct rendering

Firefox 61.0.2 on MacOS 10.14 Adobe InDesign CC Version 13.1 on MacOS 10.14
