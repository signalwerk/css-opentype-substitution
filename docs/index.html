Bug Report

CSS Styling changes OpenType substitution

Problem

A string has two characters:

A B

If the characters print next to each other the first character is replaced by the ligature feature of the font. See OpenType Docs (Microsoft) – Registered features.

AB

If the string gets formated in CSS with 

            
              p::first-letter { color: red; }
the result looks like:

AB

Expected:

—B
